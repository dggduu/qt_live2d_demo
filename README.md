## Qt_Live2D_demo
这是一个基于QT6编写的live2d桌宠。  
目前还有很多需要完善的地方。（有时间会慢慢完善~）
目前编程水平比较菜，以后可能会重构。   
**编译环境：QT 6.8.2 LLVM+Mingw64(debug)**  
*release需自行编译所需依赖库，模型放入./Resources/model/*  
*如若需直接使用https,请自行编译openssl库*
